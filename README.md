## 티스토리 블로그 스킨 메이커

### 프로젝트 개요

클로드 코드 활용 티스토리 블로그 스킨을 만들었던 저장소 입니다.

기본적으로 TISTRORY_skin_guide.md 을 바탕으로 스킨을 제작하려고 하였고, 해당 문서의 토큰이 너무 커서 주요 내용만 뽑아낸 TISTORY_quick_reference.md 를 만들었습니다.

CLAUDE.md 문서를 통해 TISTORY_quick_reference.md 를 활용하여 스킨을 제작할 수 있게 구성되어 있습니다.


**src** 폴더 내에 작업 결과물들이 버전별로 들어 있도록 구성했습니다.

**reference** 폴더에는 티스토리 기본 스킨들 예시를 넣어두었습니다.


### 수정
기본적으로 TISTORY_quick_reference.md 바탕으로 클로드 코드가 알아서 스킨을 뽑아야 하는데, 에러와 오류가 많습니다.

필요한 부분을 찾아 일일히 수정하는 작업이 필요한데, 클로드 코드가 이해를 못하는 경우 reference에 있는 스킨 기반으로 확인 후 수정 시 오류 수정 확률이 높습니다.

레퍼런스 폴더는 필요한 만큼 더 늘려도 좋고, 늘린 경우 어떤 부분을 참고해야하는지 정리된 md 파일을 하나 만드는 것을 권유 드립니다.


### 고도화
추후 위 내용을 기반으로 필요한 내용을 주고 받으면서 스킨을 뽑아내는 챗봇으로 고도화 할 예정 입니다.
